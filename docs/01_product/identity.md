# Product Identity

## Vision

<!-- プロダクトが実現したい世界観 -->

> 小売業者が地理的データと競合情報に基づき、自信を持って出店・競争戦略を決定できる世界を実現する。

## Mission

<!-- ビジョン実現のために何をするか -->

> 直感的な地図インターフェースで商圏分析と競合監視を誰でも簡単に実行可能にする。

## Principles

<!-- プロダクト開発の原則 -->

1. **ユーザー第一**: ユーザーの課題解決を最優先
2. **シンプル**: 複雑さを避け、本質に集中
3. **品質**: バグを出すより遅れる方がマシ
4. **透明性**: 決定理由を文書化し、共有する

## Positioning

<!-- 市場での位置づけ -->

| 観点             | 内容                                                                                  |
| ---------------- | ------------------------------------------------------------------------------------- |
| ターゲット       | 小売チェーン経営者、フランチャイジー、店舗展開担当者                                  |
| 課題             | 出店判断に必要な商圏・競合データの収集と分析が煩雑で、データが散在している              |
| 解決策           | 地図ベースで自店舗・競合店・商圏・人口動態を一元管理し、データドリブンな出店判断を支援 |
| 競合との差別化   | DDD + Clean Architecture による拡張性、API-first 設計、地理空間データの統合分析         |
