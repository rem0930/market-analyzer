# UI Requirements

UIの要件を定義します。

---

## General Requirements

### Accessibility
- WCAG 2.1 Level AA 準拠
- キーボードナビゲーション対応
- スクリーンリーダー対応

### Responsive Design
- ブレークポイント: 
  - Mobile: < 768px
  - Tablet: 768px - 1024px
  - Desktop: > 1024px

### Performance
- First Contentful Paint: < 1.5s
- Largest Contentful Paint: < 2.5s
- Cumulative Layout Shift: < 0.1
